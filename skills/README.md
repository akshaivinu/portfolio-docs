# Skills & Technologies

I approach technology as a toolset to solve product problems. I am adaptable and enjoy learning new tools, but here is the stack I am most proficient in:

## 🌐 Frontend
- **React.js & Next.js:** My go-to frameworks for building dynamic, SEO-friendly, and interactive user interfaces.
- **Tailwind CSS:** For rapid UI development with a consistent design system.
- **HTML5 / CSS3:** Strong foundational knowledge, including modern layout techniques like CSS Grid and Flexbox.

## ⚙️ Backend
- **Node.js:** For building scalable, event-driven server applications.
- **FastAPI:** Building high-performance Python APIs quickly with great inherent typing.
- **REST APIs:** Designing clear and predictable API contracts for client-server communication.

## 🗄️ Database
- **MongoDB:** For flexible, document-based data management when schemas are evolving rapidly.
- **MySQL / PostgreSQL:** For structured, relational data and scenarios requiring complex transactional querying.

## 🛠️ Tools & DevOps
- **Git & GitHub:** Version control, branching strategies, and collaboration.
- **Postman / Insomnia:** API testing and documentation.
- **Figma:** UI/UX prototyping and wireframing before writing code, ensuring the user experience is nailed down first.
