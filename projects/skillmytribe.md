# 📚 Skill My Tribe
> E-learning platform

**Tech Stack:** Next.js, Python, FastAPI

### Overview
Developed Skill My Tribe learning platform utilizing a modern React/Next.js flow paired with a performant Python backend.

### Development Highlights
* **Authentication:** Implementing role-based authentication mapping users through tiers (super admin → admin → student).
* **Analytics:** Designed and generated a super admin analytics dashboard to give insights into platform monitoring and student metrics.
