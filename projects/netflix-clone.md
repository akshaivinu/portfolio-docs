# 🎬 Netflix Clone
> Personal Project

**Tech Stack:** React, Tailwind CSS, Node.js, Express, TMDB API

### Overview
Built a Netflix-style streaming platform featuring dynamic content browsing, user authentication, and personalized watchlists.

### Development Highlights
* **UI/UX:** Developed responsive UI components including hero banners, movie carousels with horizontal scroll, modal previews, and search functionality mimicking Netflix’s layout patterns.
* **Backend Architecture:** Implemented backend API endpoints for movie data fetching (TMDB integration), user favorites, watch history tracking, and real-time content recommendations using Node.js/Express.
