# ✈️ Tour Booking Web Application 
> Mini Project

**Tech Stack:** React, Next.js, Tailwind CSS

### Overview
Created comprehensive tour booking web app using React/Next.js where users browse destinations, book complete tour packages, and select preferred bus options.

### Development Highlights
* **Interactive UI:** Developed intuitive UI for package selection, destination exploration with images/maps, and streamlined booking flow with payment simulation.
* **Responsive Design:** Implemented responsive design with Tailwind CSS ensuring smooth mobile booking experience and optimized performance.
